Exam Questions Help

Exam Questions is used to build and review the MCQ question paper for the selected exam.

  • Add Question creates one question with its answer options, correct answer, and marks.
  • Import Bulk Questions helps upload many questions together using the Excel template.
  • Edit is used to correct question text, image, options, correct answer, order, or marks.
  • Marks from questions should match the exam planning, because they affect scoring and results.
  • Correct Answer must be checked carefully before students attempt the exam.
Important: review every question, option, correct answer, image, and marks before publishing or allowing exam attempts.

Exam Questions या पेजचा उपयोग निवडलेल्या परीक्षेसाठी MCQ question paper तयार आणि तपासण्यासाठी केला जातो.

  • Add Question मधून एक question, त्याचे options, correct answer आणि marks add करता येतात.
  • Import Bulk Questions वापरून Excel template मधून अनेक questions एकाच वेळी upload करता येतात.
  • Edit मधून question text, image, options, correct answer, order किंवा marks दुरुस्त करता येतात.
  • Marks exam planning प्रमाणे असणे गरजेचे आहे, कारण त्यावर scoring आणि result अवलंबून असतात.
  • Correct Answer students exam attempt करण्यापूर्वी काळजीपूर्वक check करणे आवश्यक आहे.
Important: exam publish करण्यापूर्वी किंवा attempt साठी देण्यापूर्वी प्रत्येक question, option, correct answer, image आणि marks verify करा.

Exam Questions पेज का उपयोग चुनी हुई परीक्षा के लिए MCQ question paper बनाने और review करने के लिए किया जाता है.

  • Add Question से एक question, उसके options, correct answer और marks add किए जाते हैं.
  • Import Bulk Questions से Excel template का उपयोग करके कई questions एक साथ upload किए जा सकते हैं.
  • Edit से question text, image, options, correct answer, order या marks सही किए जा सकते हैं.
  • Marks exam planning के अनुसार होने चाहिए, क्योंकि scoring और result इसी पर depend करते हैं.
  • Correct Answer students के exam attempt करने से पहले carefully check करना जरूरी है.
Important: exam publish करने या attempt के लिए देने से पहले हर question, option, correct answer, image और marks verify करें.
