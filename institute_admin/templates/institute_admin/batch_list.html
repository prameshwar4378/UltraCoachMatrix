Batch Management Help

Batch Management is used to create and maintain divisions or sections for your institute. For example, after creating a course/class like 10th, you can create batches such as 10th-A, 10th-B, or 10th-C.

  • Total Batches, Active Batches, Inactive Batches, and Available Courses show a quick summary of your division setup.
  • Filter Batches helps you search by course/class, status, division name, course name, or timing.
  • New Batch opens the form where you can create divisions like 10th-A or 10th-B, connect them to courses/classes, assign teachers, and set dates or timing.
  • Batch Records displays linked courses/classes, total fee, timing, date range, teachers, and status for each division.
  • Edit and Delete actions help you maintain existing division data.

Batch Management चा वापर आपल्या संस्थेतील divisions किंवा sections तयार आणि व्यवस्थापित करण्यासाठी केला जातो. उदाहरणार्थ, 10th हा course/class तयार केल्यानंतर 10th-A, 10th-B किंवा 10th-C असे batches तयार करता येतात.

  • Total Batches, Active Batches, Inactive Batches आणि Available Courses मधून division setup चा झटपट आढावा मिळतो.
  • Filter Batches वापरून course/class, status, division name, course name किंवा timing नुसार batches शोधता येतात.
  • New Batch मधून 10th-A किंवा 10th-B सारखी divisions तयार करता येतात, courses/classes जोडता येतात, teachers assign करता येतात आणि dates किंवा timing set करता येते.
  • Batch Records मध्ये प्रत्येक division साठी linked courses/classes, total fee, timing, date range, teachers आणि status दिसतो.
  • Edit आणि Delete actions वापरून existing division data update किंवा remove करता येतो.

Batch Management का उपयोग आपके institute में divisions या sections बनाने और manage करने के लिए किया जाता है. उदाहरण के लिए, 10th course/class बनाने के बाद आप 10th-A, 10th-B या 10th-C जैसे batches बना सकते हैं.

  • Total Batches, Active Batches, Inactive Batches और Available Courses आपके division setup का quick summary दिखाते हैं.
  • Filter Batches से आप course/class, status, division name, course name या timing के अनुसार batches खोज सकते हैं.
  • New Batch से 10th-A या 10th-B जैसी divisions बनाई जा सकती हैं, courses/classes connect किए जा सकते हैं, teachers assign किए जा सकते हैं और dates या timing set की जा सकती है.
  • Batch Records हर division के linked courses/classes, total fee, timing, date range, teachers और status दिखाता है.
  • Edit और Delete actions से existing division data को update या remove किया जा सकता है.
