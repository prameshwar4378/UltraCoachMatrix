Print Template Center Help

The Print Template Center lets the institute preview, download, upload, and select default print designs for documents such as admission forms, certificates, and payment receipts.

  • Template groups keep designs separated by document type, so staff select the correct layout for each print document.
  • Preview helps verify the design before setting it as the active institute template.
  • Set as Template makes a selected library design the institute default for that document type.
  • Upload Custom Template is used when the institute needs its own HTML print layout.
  • Use Default removes the selected institute template and returns that document type to the built-in default.
Important: preview every template after upload or selection, because print alignment, logo size, and page margins affect official documents.
