Course Management Help

Course Management is used to create and maintain the courses or classes offered by your institute. These courses are later connected with batches, students, fees, homework, exams, and reports.

  • Total Courses, Active Courses, Inactive Courses, and Total Batches show a quick summary of your academic setup.
  • Filter Courses helps you find courses by status or course name.
  • New Course opens the course form where you can add course name, duration, fee, description, and status.
  • Bulk Create helps school institutes create standard classes quickly, such as Nursery, LKG, UKG, and Class 1 to Class 10.
  • Edit and Delete actions are available in the course records table for maintaining existing course data.

Course Management चा वापर आपल्या संस्थेतील courses किंवा classes तयार आणि व्यवस्थापित करण्यासाठी केला जातो. हे courses पुढे batches, students, fees, homework, exams आणि reports सोबत जोडले जातात.

  • Total Courses, Active Courses, Inactive Courses आणि Total Batches मधून academic setup चा झटपट आढावा मिळतो.
  • Filter Courses वापरून status किंवा course name नुसार courses शोधता येतात.
  • New Course मधून course name, duration, fee, description आणि status भरून नवीन course तयार करता येतो.
  • Bulk Create school institutes साठी Nursery, LKG, UKG आणि Class 1 ते Class 10 सारखे standard classes पटकन तयार करण्यास मदत करते.
  • Edit आणि Delete actions वापरून existing course records update किंवा remove करता येतात.

Course Management का उपयोग आपके institute में available courses या classes बनाने और manage करने के लिए किया जाता है. ये courses आगे batches, students, fees, homework, exams और reports से connect होते हैं.

  • Total Courses, Active Courses, Inactive Courses और Total Batches आपके academic setup का quick summary दिखाते हैं.
  • Filter Courses से आप status या course name के अनुसार courses खोज सकते हैं.
  • New Course से course name, duration, fee, description और status भरकर नया course बनाया जा सकता है.
  • Bulk Create school institutes को Nursery, LKG, UKG और Class 1 से Class 10 तक standard classes जल्दी बनाने में मदद करता है.
  • Edit और Delete actions से existing course records को update या remove किया जा सकता है.
