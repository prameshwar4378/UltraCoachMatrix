Premium Report Management Help

The Premium Report Management Dashboard brings finance, fee, student, and academic reports into one place so institute staff can quickly choose the right report and understand what each output will contain.

  • Report groups separate reports by purpose, such as finance, student records, and academic performance.
  • Filters show which inputs will be needed before generating a report, including dates, class, batch, student, and status.
  • Output chips show the important columns or summaries expected in each generated report.
  • Generate opens reports that are already connected, while Planned marks reports that are not active yet.
  • Saved reports will be used to track generated files and make repeated report access easier.
Important: always check the selected date range, class, batch, and student filters before exporting a report for official use.
